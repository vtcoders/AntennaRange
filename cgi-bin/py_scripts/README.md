## Running and Compiling
### Application
Ensure you have ```python3``` and ```pymongo``` <br>
```
pip3 install pymongo[srv]
```

Running User Database <br>
```
python3 userDB.py
```

Running Experiment Database <br>
```
python3 expDB.py
```
